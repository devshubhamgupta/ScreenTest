# ScreenTest - Mobile Device Simulator Chrome Extension

**ScreenTest** is a powerful Chrome extension that allows developers and designers to test websites on 80+ mobile devices instantly. Preview your website on realistic iPhone, Android, tablet, and special device frames with accurate bezels, notches, and status bars.

## üöÄ Features

- **80+ Device Presets**: Test on iPhone (5 to 17 Pro Max), Android phones (Galaxy, Pixel, Xiaomi, etc.), tablets (iPad, Galaxy Tab), and special devices (MacBook, iMac, Smart TV, etc.)
- **Realistic Device Frames**: Beautiful device frames with accurate bezels, notches (iPhone X-14), Dynamic Island (iPhone 14 Pro+), and punch-hole cameras (Android)
- **Portrait & Landscape Modes**: Toggle between portrait and landscape orientations with one click
- **Auto-Scaling**: Automatically scales devices to fit your viewport
- **Status Bars**: Realistic status bars with time, signal, WiFi, and battery indicators
- **Device Information**: Shows device name and dimensions above each frame
- **Beautiful UI**: Modern orange-themed interface with animated background and grid pattern
- **Zero Configuration**: Works instantly - just click the extension icon and start testing

## üîß Installation

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the `dist` folder
5. The ScreenTest icon will appear in your Chrome toolbar

## üìñ How to Use

1. Click the ScreenTest extension icon in Chrome
2. The simulator opens with the current tab's URL (or default URL)
3. Click the device icon in the sidebar to select from 80+ devices
4. Click the rotate icon to toggle between portrait and landscape
5. The device automatically scales to fit your viewport
